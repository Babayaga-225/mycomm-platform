# MyComm Platform — Site vitrine

Site officiel de **MyComm Platform**, agence de transformation digitale basée à Bouaké, Côte d'Ivoire.

Fondateur : Anicet Tafotien Koné — consultant en transformation digitale.

## Contenu

Site statique (HTML/CSS/JS, animations GSAP). Aucune dépendance de build.

- `index.html` — la page complète du site
- `mycomm-logo.png`, `mycomm-icon.png` — identité visuelle
- `Logo_Connect_Chill.png` — logo du premier client (témoignage)

## Aperçu en local

Ouvrir `index.html` dans un navigateur. Pour que les images locales se chargent
correctement en `file://`, lancer Chrome avec le drapeau
`--allow-file-access-from-files`, ou servir le dossier via un petit serveur statique.

## Mise en ligne

Hébergement statique : déposer le contenu du dossier dans `public_html`
(Hostinger / OVH) ou sur tout hébergeur de fichiers statiques.
